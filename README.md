## Featured Projects

### Multimodal Learning Research

Undergraduate research on image captioning, visual question answering, multimodal retrieval, representation learning, and cross-modal alignment.

### Facial Expression Analysis

Studied FaceXFormer and POSTER, then re-implemented key ideas for transformer-based facial expression recognition.

### Vietnamese Text Classification

Re-implemented a multi-channel LSTM-CNN architecture and adapted it from Vietnamese sentiment analysis to Vietnamese news topic classification.

### YouTube Music / Data Dashboard

Built data workflows and AI-agent-assisted dashboard features for exploring a self-collected YouTube music dataset.

Explored OCR and image preprocessing for a difficult angled Vietnamese license plate case using OpenCV, Colab, YOLO experiments, and PaddleOCR-oriented pipelines.
